A plugin to add customizable floating social buttons to your WordPress site.

== Description ==

The Floating Social Buttons Widget plugin allows you to add and customize floating social buttons on your WordPress site. You can select the position, style, and size of the buttons, as well as customize the main button icon and its colors.

=== Features ===
* Choose from various button styles (Circle, Square, Rounded).
* Select the position of the buttons (bottom right, bottom left).
* Customize the main button icon and colors.
* Add multiple social buttons with individual icons, colors, and links.
* Use any social network icon from the open source icon library "Simple Icons" (https://https://simpleicons.org)

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/floating-social-buttons-widget` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to the 'FSBw' settings page to configure the plugin.

== Frequently Asked Questions ==

= How do I add a new social button? =

Go to the 'FSBw' settings page, and click the 'Add Button' button. Fill in the icon name, button color, icon color, and add your link.

= How do I customize the main button icon? =

In the 'FSBw' settings page, you can enter the SVG code for the main button icon in the 'Main Button Icon' field. You can also select the main button color and icon color.
